# Security Policy

## Reporting Security or Safety Issues

Please do not open public issues for severe vulnerabilities, unsafe profile behavior, prompt-injection paths, or safety bypasses.

Report issues privately to:

```text
security@example.com
```

Include:

- A concise description of the issue.
- A minimal reproduction if possible.
- Affected versions or commit hashes.
- Any safety impact, including whether the issue can produce abuse, threats, hate, manipulation, or unsafe crisis behavior.

## Safety-Sensitive Changes

Changes to `SafetyPolicy`, built-in profiles, adapters that construct prompts, and documentation that defines safe usage require additional review.

## Supported Versions

The MVP currently supports the latest `main` branch and the latest released package version.
